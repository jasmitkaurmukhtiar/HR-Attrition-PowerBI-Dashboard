# HR Attrition Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard created to analyze employee attrition within an organization. It provides insights into workforce trends by examining attrition across departments, job roles, age groups, gender, salary, and other employee attributes. The dashboard is designed to help HR teams identify patterns and support data-driven retention strategies.

---

## Objective

The main objective of this project is to:

- Analyze employee attrition across different departments and job roles.
- Identify factors that may contribute to higher employee turnover.
- Understand the relationship between employee demographics and attrition.
- Present HR data in a clear, interactive dashboard for better decision-making.

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset

---

## Key Metrics

- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Monthly Salary

---

## Dashboard Features

- Department-wise attrition analysis
- Job role-wise attrition analysis
- Gender distribution
- Attrition across different age groups
- Salary vs. Age analysis
- Department contribution to overall attrition

---

## Key Insights

- The Sales department recorded the highest attrition rate.
- Some job roles experienced noticeably higher employee turnover than others.
- Employees between the ages of 30 and 40 accounted for a significant portion of attrition.
- The dashboard helps highlight trends that can support employee retention and workforce planning.

---

## Future Improvements

## Dataset

The dashboard uses the IBM HR Analytics Employee Attrition dataset, which contains employee demographic information, job details, compensation, and attrition records.
